# AI Policy for Course Work

## Purpose

This document outlines the acceptable use of AI tools in this course.

## Acceptable Use

Students may use AI tools such as:
- Code completion tools (GitHub Copilot, etc.)
- ChatGPT, Claude, or similar language models for:
  - Understanding concepts
  - Debugging assistance
  - Code explanation
  - Learning new syntax or techniques

## Requirements

When using AI tools:

1. **Attribution**: Always acknowledge when AI tools were used in your work
2. **Understanding**: You must understand any code or content generated by AI
3. **Verification**: Verify all AI-generated content for accuracy
4. **Original Work**: Your submissions should reflect your own understanding and analysis

## Prohibited Use

AI tools should NOT be used for:
- Generating entire assignments without understanding
- Copying AI-generated code without modification or understanding
- Bypassing learning objectives
- Academic dishonesty

## Disclosure

Include a brief statement in your assignments describing how you used AI tools, for example:
- "I used ChatGPT to help debug an error in my R code"
- "I used GitHub Copilot for code completion suggestions"
- "I used Claude to help understand the difference between merge and join operations"

## Questions

If you're unsure whether a specific use of AI is appropriate, please ask the instructor.
