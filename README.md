# Class Repository

This is a structured repository for course materials, data, and analysis.

## Repository Structure

```
├── data/           # Datasets and data files
├── analysis/       # Analysis scripts and notebooks
├── shiny_app/      # Test Shiny application
├── AI_POLICY.md    # Guidelines for AI tool usage
└── README.md       # This file
```

## Getting Started

1. **Data**: Place your datasets in the `data/` folder
2. **Analysis**: Store your analysis scripts in the `analysis/` folder
3. **Shiny App**: A test Shiny application is available in `shiny_app/`
4. **AI Policy**: Review the AI_POLICY.md for guidelines on using AI tools

## Requirements

For the Shiny app:
- R (>= 4.0.0)
- shiny package

Install required packages:
```r
install.packages("shiny")
```

## Running the Shiny App

```r
shiny::runApp("shiny_app")
```
